# Береза Найф — CRM контекст

## Стек
- Чистый HTML/JS, один файл index.html
- Хостинг: Netlify (auto-deploy из GitHub bereza-knife/finance_bereza)
- Данные: localStorage (временно, план — Google Sheets)

## Команда
- Лёша — производство
- Миша — продажи  
- CFO (ты) — финансы

## Структура артикула
модель - сталь - больстер - вставка - ручка - форма - лайнер - всечка
Пример: 023-05-01-04-01

## Финансовая логика
- Себес в CRM = реальный себес × 1.3 (наценка скрыта)
- Маржа считается от этой цифры
- Схема 50/50: трансфертная цена = себес + 50% от (розница - себес)
